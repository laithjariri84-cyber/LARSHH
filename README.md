# LARSSH

**AI Real Estate Intelligence Platform**

Premium AI-powered real estate platform for property intelligence, market analytics, and advisory workflows.

## Stack

- **Next.js 15** · React 19 · TypeScript
- **PostgreSQL** (Supabase) · Prisma
- **Tailwind CSS** — black & gold design system

## Getting started

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Documentation

See [docs/README.md](./docs/README.md) for architecture, database design, and product specs.

## Environment

Set `NEXT_PUBLIC_APP_NAME=LARSSH` in `.env` for client-side branding.
